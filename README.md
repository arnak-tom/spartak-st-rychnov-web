# Spartak ST Rychnov - Web

Web oddílu mládeže stolního tenisu **Spartak Rychnov u Jablonce nad Nisou**.

## O projektu

Tento web byl vytvořen jako edukativní projekt s cílem:
- ✅ Prezentovat oddíl mládeže stolního tenisu
- ✅ Poskytnout informace o trénincích a kontaktech
- ✅ Vytvořit příklad projektu pro děti základní školy
- ✅ Demonstruje vývoj webové aplikace krok za krokem

## Technologie

- HTML5
- CSS (LESS)
- JavaScript / TypeScript
- Firebase Firestore (později)

## Struktura projektu

```
spartak-st-rychnov-web/
├── index.html          # Hlavní stránka
├── styles/
│   └── style.less      # Stylování (LESS)
├── js/
│   └── main.js         # JavaScript
├── README.md           # Tento soubor
├── VYVOJ.md            # Dokumentace vývoje (pro učení)
└── .gitignore          # Git ignore soubor
```

## Jak spustit

1. Klonujte repozitář
2. Otevřete `index.html` v prohlížeči
3. (později) Nastavte Firebase konfiguraci
