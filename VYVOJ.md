## 🚀 KROK 1: Vytvoření repozitáře a základní struktura

### Co jsme udělali:

#### 1.1 Vytvořili jsme repozitář na GitHubu
- Přešli jsme na https://github.com/new
- Vyplnili jsme název: `spartak-st-rychnov-web`
- Nastavili jsme jako **Public** (aby byl vidět pro všechny)
- Přidali jsme README soubor
- Vytvořili jsme `.gitignore` pro Node.js projekty

**Proč to děláme?**
- GitHub je místo, kde si můžeme ukládat kód
- Kdokoliv může vidět, jak jsme web dělali
- Můžeme si historii změn prohlédnout

#### 1.2 Aktualizovali jsme README.md
- README je soubor, který se zobrazuje na GitHubu
- Obsahuje základní informace o projektu
- Pomáhá ostatním pochopit, co projekt dělá

#### 1.3 Vytvořili jsme tento soubor (VYVOJ.md)
- Je to průvodce pro učení
- Ukazuje, jak jsme projekt vytvářeli
- Můžeš jej následovat a vytvořit si podobný web

### Soubory vytvořené v KROKU 1:
- ✅ `README.md` - informace o projektu
- ✅ `VYVOJ.md` - tento soubor (průvodce)
- ✅ `.gitignore` - soubor, který říká Gitu, které soubory ignorovat

### Co bude dál:

V příštím kroku vytvoříme:
- `index.html` - hlavní HTML stránka
- `styles/` složka s `style.less` - stylování
- `js/` složka s `main.js` - JavaScript kód

## 📚 Slovníček pojmů

- **GitHub** - služba pro ukládání kódu a správu verzí
- **Repozitář** - složka na GitHubu, kde je uložen váš projekt
- **HTML** - jazyk pro tvorbu webových stránek
- **CSS** - jazyk pro stylování webových stránek
- **LESS** - CSS preprocesor, který dělá psaní CSS jednodušší
- **JavaScript** - programovací jazyk pro webové stránky
- **Firebase** - služba od Googlu pro ukládání dat a autentizaci
- **Mobile-first** - přístup, kdy nejdřív navrhujeme pro mobilní zařízení
